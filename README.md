## 技术栈  
vite + vue3 + vue-router + pinia + axios + element-plus 

## 更新包  
1. 安装npm-check-updates
  ```
  npm install -g npm-check-updates
  ```
2. 运行 ncu 来显示要升级的软件包
  ```
  ncu
  ```
3. 全量更新所有包至最新版本
  ```
  ncu --upgrade
  // or 
  ncu -u
  ```
4. 或者手动选择需要更新的包
  ```
  ncu --interactive
  // or 
  ncu -i
  ```
